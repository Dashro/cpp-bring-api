# Bring Shopping Lists API

An unofficial C++/Qt package to access the Bring! shopping lists API.

> This is a **minimal** C++/Qt port of the [node-bring-api](https://github.com/foxriver76/node-bring-api) by [foxriver76](https://github.com/foxriver76). All credit goes to him for making this awesome API possible!
